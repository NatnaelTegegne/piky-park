# Pikipark
## the easy parking solution for both businesses and drivers
Created in under 24 hours for [NexHacks 2026](https://nexhacks.devpost.com/)

### Description
A map application that provides the user detailed parking space availability in specific locations using an AI model trained with Machine Learning
